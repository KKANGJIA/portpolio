aT 농식품 빅데이터 기반 온라인 해커톤

1. 프로젝트 취지: 6차 산업을 농촌체험과 연관하여 농촌의 성장과 경쟁력을 확보
2. 개발 기간: 1~2주일
3. 사용기술: Java, HTML, CSS, Python, Machin Leaning, DB, Google API, Git & GitHub 등
4. 팀원: 프로그래머 3명(백엔드 1명, 프론트엔드 1명, 빅데이터 1명)
5. 나의 역할: 프로그래머 - 프론트엔드
6. 플랫폼: 웹사이트

프로젝트 구성

1. 농촌체험 예약: 농촌체험 인기순위, 농촌체험 기업 상세안내, 예약확인으로 구성

2. 체험위치 검색: 구글 지도를 이용하여 체험 위치를 쉽게 파악

3. 리뷰 검색: 다른 사용자들의 체험리뷰를 확인하고 체험 후 리뷰를 작성

개발 과정

1. kakao oven을 통한 프로토 타입 제작

2. 실제 개발(by Visual Studio Code Editor)
